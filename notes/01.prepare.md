# Prepare

## Node Version

node v22

## Install pnpm

```zsh
# install corepack
npm install -g corepack@latest

# enable pnpm
corepack enable pnpm

# where pnpm installed
which pnpm # /Users/kvoon/.nvm/versions/node/v22.14.0/bin/pnpm
```

## Nuxt 4

```ts
export default defineNuxtConfig({
```
